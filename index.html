Arabic/Farsi Language Classifier

This project was inspired by the Python langid module. This is a generic language classifier that uses the frequency of 3 character sequences as features for a Naive Bayes classifier.

Method

I wanted to assess to what extent the 3-character feature space could distinguish between two languages that are often confused.
